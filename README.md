# www.motowear.gr

## Docker Stack Services

- **dozzle**: Real-time container logs viewer
- **postgres**: PostgreSQL database
- **pgweb**: Web-based UI for PostgreSQL
- **minIO**: S3-compatible object storage for product images
- **typesense**: Blazing fast search engine (alternative to ElasticSearch)
- **next.js**: Full-stack application (frontend + backend)
- **nginx**: Reverse proxy for routing and SSL termination
- **docker stack**: Manages service orchestration and load balancing
